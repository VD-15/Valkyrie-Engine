Detailed Description

Represents a dynamic event sent to an entity, Unlike events, which are statically defined and sent globally via the EventBus, signals are completely dynamic and are sent only to the Entity that recieves them. The components that are attached to the entity then process the signal by inheriting from ISignalListener.
