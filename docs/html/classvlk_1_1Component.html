Detailed Description

template<typename T>
class vlk::Component< T >

Template class for ECS components.

Components are storage objects that define little to no logic. Components can be attached to entities with Attach() to provide the entity with some sort of property. A Component attached to an entity can then be fetched using FindOne(EntityID) or FindMany(EntityID). Components can also be accessed or modified by passing a function object to either ForEach(std::function<void(const T*)>) or ForEach(std::function<void(T*)>)

Some behaviour related to Components can be controlled by specializing GetComponentHints() for T.

Template Parameters
TThe data this entity is storing. Ideally this would be a POD struct, but any type with at least one public constructor and a public destructor will work.
