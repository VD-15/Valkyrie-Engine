static void vlk::EventBus< T >::AddListener ( IEventListener< T > *  listener)
inlinestatic

Adds an event listener to the event bus.

Once added, an event listener will then have IEventListener<T>::OnEvent(const T&) called whenever an event is sent. If the listener is already present, it is not added again. The engine makes no guarantee as to what order event listeners are called in.

Thread Safety
May be called from any thread.
Resource locking is handled internally.
Unique access to this class is required.

template<typename T >
static void vlk::EventBus< T >::Send ( const T &  t)
inlinestatic

Raises the IEventListener<T>::OnEvent callback for every IEventListener present in the bus.

All listeners get called immediately, one after the other, on the calling thread. This function returns once all listeners have finished executing.

Thread Safety
May be called from any thread.
Resource locking of this class is handled internally.
Event listeners must implement their own resource locking.
Shared access to this class is required.
This function may block the calling thread.
