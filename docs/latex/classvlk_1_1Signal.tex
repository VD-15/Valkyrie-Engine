\doxysubsection{Detailed Description}
Represents a dynamic event sent to an entity, Unlike events, which are statically defined and sent globally via the \mbox{\hyperlink{classvlk_1_1EventBus}{Event\+Bus}}, signals are completely dynamic and are sent only to the \mbox{\hyperlink{classvlk_1_1Entity}{Entity}} that recieves them. The components that are attached to the entity then process the signal by inheriting from \mbox{\hyperlink{classvlk_1_1ISignalListener}{I\+Signal\+Listener}}.
