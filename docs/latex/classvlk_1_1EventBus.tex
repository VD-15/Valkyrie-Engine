Adds an event listener to the event bus. 

Once added, an event listener will then have \mbox{\hyperlink{classvlk_1_1IEventListener_a927a7b7ac1c1fb665184554d43160c7f}{IEvent\+Listener$<$\+T$>$\+::\+On\+Event(const T\&)}} called whenever an event is sent. If the listener is already present, it is not added again. The engine makes no guarantee as to what order event listeners are called in.
