\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class vlk\+::\+Component$<$ T $>$}

Template class for ECS components. 

Components are storage objects that define little to no logic. Components can be attached to entities with \mbox{\hyperlink{classvlk_1_1Component_a907b54959d286308b0db84136c58a631}{Attach()}} to provide the entity with some sort of property. A \mbox{\hyperlink{classvlk_1_1Component}{Component}} attached to an entity can then be fetched using \mbox{\hyperlink{classvlk_1_1Component_a2981e217cc4409b3f4e06fcc52d2a0fb}{Find\+One(\+Entity\+ID)}} or Find\+Many(\+Entity\+ID). Components can also be accessed or modified by passing a function object to either For\+Each(std\+::function$<$void(const T$\ast$)$>$) or For\+Each(std\+::function$<$void(\+T$\ast$)$>$)

Some behaviour related to Components can be controlled by specializing \mbox{\hyperlink{Component_8hpp_ac5474f9273bb9e982abb8e53cd4a34f9}{Get\+Component\+Hints()}} for T.
